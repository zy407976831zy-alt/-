## 当前项目定位

这个项目不是普通 AI 生图工具，而是一个面向婚纱摄影工作室内部使用的影像生产台。

核心目标：

- 上传客户原片。
- 上传一张参考图。
- 先分析参考图的风格、场景、光影、色调、服装、发型、动作、机位和焦距。
- 再按统一风格批量生成客户照片。
- 必须保留客户本人身份，不能换脸。
- 皮肤要高端商业精修，但必须保留真实毛孔和肤质。
- 输出结果要拒绝 AI 感、贴图感、一眼假。
- 结果需要可下载高质量 JPG。
- 后续准备部署到云端，给员工直接使用。

## 已接入能力

### 1. OpenAI API 图像生成

当前已接入 OpenAI 图像生成接口。

环境变量：

```text
OPENAI_API_KEY=你的 API Key
OPENAI_IMAGE_MODEL=gpt-image-2
OPENAI_VISION_MODEL=gpt-4.1-mini
```

当前 `.env` 已在本地配置，不能上传到 GitHub，不能发给员工。

### 2. 参考图分析

参考图分析已经接入视觉模型，不再是固定模板。

当前要求模型分析：

- 整体商业风格。
- 场景空间。
- 人物动作表情。
- 服装造型。
- 发型和头纱。
- 光源类型，柔光/硬光，主光方向，阴影边缘。
- 色彩层次，高光、中间调、暗部、肤色、背景灰阶。
- 商业后期方式。
- 机位、构图、焦距、景别、拍摄距离、人物占比、裁切边界、空间纵深。
- 参考图视觉重点，例如头纱、裙摆、花束、手部动作、特殊道具。

### 3. 生成控制选项

当前网页已有中文选项：

- 处理方式：参考图改风格 / 只精修原片
- 发型参考：不参考 / 参考风格 / 匹配参考
- 背景参考：参考风格 / 强匹配参考 / 不参考
- 服装参考：参考风格 / 不参考 / 匹配参考
- 动作表情：动作保持 / 小幅调整 / 重新设计
- 生成张数：1-4
- 画幅比例：横图 3:2 / 竖图 2:3 / 方图 1:1
- 身形精修：自然优化 / 自然保持 / 适度显瘦
- 原片缺陷优化：自然修正 / 严格保留 / 明显修正
- 精修强度：高端精修 / 商业精修 / 轻精修
- 色彩后期：高端调色 / 强匹配参考 / 自然校色

### 4. 人物身份锁定

当前提示词已强化身份锁定。

重点要求：

- 客户原片是唯一身份来源。
- 不换脸。
- 不生成参考图里的人。
- 不生成 AI 美女模板脸。
- 保留眼型、眼距、鼻子、人中、嘴唇、脸宽脸长、下颌、颧骨、痣点、年龄感和个人气质。
- 可以修正手机自拍、广角、近距离导致的头部拉长、脸部压缩、肩颈变形，但只能恢复自然比例，不能换脸。

### 5. 批量生成

当前上传客户原片支持多选。

使用方式：

1. 上传多张客户原片。
2. 上传 1 张参考图。
3. 点击“分析参考图”。
4. 设置统一选项。
5. 点击“生成图片”。
6. 系统会按顺序逐张生成。

如果上传 10 张照片，生成张数选 2，则会生成 20 张结果。

当前批量逻辑是顺序生成，速度相对稳定但较慢。后续可增加“同时生成数量：1 / 2 / 3”来提高批量速度。

### 6. 高质量 JPG 下载

生成结果会保存：

- `result.png`
- `result.jpg`

网页可下载高质量 JPG。

为了云端兼容，已经把原本依赖 Mac `sips` 的转换逻辑改成优先使用 `sharp`。

### 7. 成片交付面板

原来的“助手质检”已改为“成片交付”。

当前包含：

- 当前状态：待生成 / 生成中 / 待审核 / 可交付 / 需重做 / 废片
- 当前版本
- 智能审核
- 可交付
- 重做
- 重做原因快捷按钮：脸不像、背景不像、发型未匹配、光影不对、色调不对、贴图感、手部问题、比例失衡
- 内部备注
- 下载高质量 JPG
- 复制备注

质检红线已经改成后台默认全部启用，不再需要员工手动勾选。

## 当前已知问题与后续优化建议

### 1. 生成速度

当前批量生成是顺序执行，速度较慢。

建议后续增加：

```text
同时生成数量：1 / 2 / 3
```

默认 1 最稳定，2-3 可提高批量效率。

### 2. 云端存储

正式部署后需要对象存储。

否则员工长期使用时，历史客户图和成片不够安全稳定。

### 3. 登录权限

部署到公网前，需要增加登录密码或账号系统。

否则任何拿到链接的人都可能消耗 API 额度。

### 4. 批量结果管理

后续可优化：

- 每张图单独状态。
- 每张图单独下载。
- 一键下载全部 JPG ZIP。
- 批量失败重试。
- 版本对比。
- 标记可交付。

### 5. 页面视觉继续细化

当前已进入“高端摄影工作室官网 / 杂志编辑部风”方向。

后续如继续优化，可重点调整：
