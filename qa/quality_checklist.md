# 婚纱照生成质检清单

## 红线

任意一项出现，不能给客户看：

- [ ] 脸不像本人。
- [ ] 发型在未授权时被参考图污染。
- [ ] 服装在未授权时被参考图强行替换。
- [ ] 人物像贴在背景上。
- [ ] 一眼看出 AI 感。
- [ ] 皮肤像塑料或蜡像。
- [ ] 光影、色温、阴影明显不统一。
- [ ] 人物胖瘦、头身比例或画面大小明显失衡。
- [ ] 脸像本人，但身体、光影、精修或融合质量达不到客片标准。
- [ ] 手脚、婚纱、发丝有明显畸变。
- [ ] 动作表情改变后不像本人或人体不自然。
- [ ] 图片会损害门店专业度和客户信任。

## A. 身份一致性

- [ ] 客户脸型没有变窄、变尖、变圆或变年轻过度。
- [ ] 眼睛大小、眼距、眼型和眼神特征一致。
- [ ] 鼻梁、鼻头、鼻翼没有被重新生成。
- [ ] 嘴型、唇峰、嘴角和表情特征一致。
- [ ] 下颌线、颧骨、额头比例和原片一致。
- [ ] 没有变成参考图里的人。
- [ ] 未开启发型参考时，发型主体和原片一致，没有复制参考图刘海、发色、头饰。
- [ ] 开启发型参考时，只改变被允许的发型方向，没有影响脸型、发际线、年龄感和身份。
- [ ] 表情气质和原片一致，没有变成 AI 美女模板脸。

结论：

- [ ] 通过
- [ ] 返工
- [ ] 废片

## B. 皮肤商业精修

- [ ] 皮肤干净，但仍保留毛孔和真实肌理。
- [ ] 没有塑料磨皮、蜡像感、假白。
- [ ] 面部高光和阴影符合场景光线。
- [ ] 颈部、手臂、腿部肤色与面部统一。
- [ ] 有商业修图的干净度和明暗塑形，不是未处理的原片质感。
- [ ] 皮肤、妆面、肩颈、手臂、腿部的精修程度统一。
- [ ] 妆面自然，未改变客户身份。

结论：

- [ ] 通过
- [ ] 返工
- [ ] 废片

## C. 场景融合

- [ ] 人物与背景透视一致。
- [ ] 人物在画面中的大小符合场景空间，不显得过大、过小、拉长或压扁。
- [ ] 人物胖瘦、肩颈、腰臀、四肢长度没有因为换场景而失真。
- [ ] 人物脚底、身体、婚纱与家具/地面有真实接触阴影。
- [ ] 光线方向和参考场景一致。
- [ ] 婚纱边缘、头纱、发丝没有明显穿帮。
- [ ] 背景不抢脸，主体明确。
- [ ] 人物没有漂浮感。
- [ ] 人物边缘没有抠图白边、黑边、光晕。
- [ ] 背景不是虚假的 AI 贴片，具备真实摄影空间感。

结论：

- [ ] 通过
- [ ] 返工
- [ ] 废片

## C2. 真实摄影感

- [ ] 画面像真实相机拍摄，而不是 AI 合成图。
- [ ] 景深、噪点、锐度、对比度统一。
- [ ] 皮肤、婚纱、背景的清晰度等级一致。
- [ ] 没有局部过锐、局部糊成一片的问题。
- [ ] 没有不合理的高光、反光、阴影断层。
- [ ] 人物边缘没有抠图白边、黑边或光晕。
- [ ] 脚下、裙摆、手臂、婚纱与环境有真实接触阴影。
- [ ] 人物和背景的颗粒、空气感、镜头质感一致。
- [ ] 画面没有过度梦幻、过度通透、过度 AI 高亮。

结论：

- [ ] 通过
- [ ] 返工
- [ ] 废片

## D. 人体与婚纱细节

- [ ] 手指数量正确，关节自然。
- [ ] 腿部、手臂、肩颈比例自然。
- [ ] 头身比例、身体厚度、腰臀比例和原片身份一致。
- [ ] 身形有商业修饰，但没有变成另一个人的身材。
- [ ] 未开启动作表情改变时，原片动作和表情没有被重做。
- [ ] 开启动作表情改变时，新动作自然、符合婚纱摄影姿态、没有破坏身份。
- [ ] 婚纱结构合理，没有融化、断层、脏边。
- [ ] 头发自然，不糊、不成块、不穿脸。
- [ ] 未开启服装参考时，服装没有被强行替换成参考图款式。
- [ ] 开启服装参考时，服装变化合理，没有破坏身体比例、遮挡关系和真实布料结构。

结论：

- [ ] 通过
- [ ] 返工
- [ ] 废片

## E. 最终交付

- [ ] 适合给客户看。
- [ ] 适合进入人工精修。
- [ ] 文件命名正确。
- [ ] 已记录版本和返工原因。
- [ ] 不会让客户质疑“这是不是假图”。
- [ ] 不会损害门店品牌和信任。
