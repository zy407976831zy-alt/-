# 员工操作网页原型

## 当前版本能力

这是第一版本地网页原型，用来跑通新的生产思路：

`参考图分析风格 -> 客户原片应用风格 -> 生成提示词 -> 复制到出图平台`

当前支持：

- 上传客户原片预览
- 上传参考图预览
- 选择风格包
- 选择发型参考、服装参考、动作表情、生成张数
- 自动生成中文长提示词
- 复制提示词
- 下载提示词 `.md` 文件
- 预留 Image 2.0 生成按钮
- 模拟生成结果
- 助手质检占位
- 标记通过/返工

## 当前版本不做什么

当前还没有真实后端，因此还没有真正接入 Image 2.0、GPT 图像生成、ComfyUI 或其他出图平台。

现阶段操作方式：

1. 打开 `web/index.html`。
2. 上传客户原片和参考图用于记录。
3. 选择中文选项。
4. 复制生成提示词。
5. 如果后端未接入，先到已经跑通的 GPT/Image 2.0 出图入口上传客户原片并粘贴提示词。
6. 如果后端已接入，直接点击“生成图片”。
7. 在网页里做质检和返工记录。
